URL Shortener Help

Introduction

This is the help page for the URL Shortener add-on for Google Docs, which is used to shorten long URLs using the Google URL Shortener service.

Using the Add-on

Getting the Document URL

Google Drive documents typically have rather long URLs. You can use the URL Shortener add-on to get a shortened link to your active document by selecting Add-ons > URL Shortener > Get document link.

Shortening a URL within a Document

To shorten a URL in your document:
  1. Select the URL you want to shorten.
  2. Click Add-ons > URL Shortener > Shorten selected URL.
  3. Click Replace link to replace the long URL with the shortened version.
  4. Click Close to return to your document.

Invalid URLs

The following kinds of URLs are considered invalid, and cannot be shortened:
  • URLs containing spaces
  • URLs containing backslashes
  • URLs containing no periods
  • URLs that have already been shortened
